Overview

Provides a mechanism for parsing geolocations from strings, calculating distances between geolocations, and formatting human-readable distances.

+ distance:

Calculates the distance between two map coordinates, in miles.

This function accepts an array of template expressions, and expects two objects in the array: point a and point b, both of which are expected to evaluate to MBGeolocation instances or NSStrings. If either parameter is a string, an attempt is made to convert it into an MBGeolocation instance by interpreting it in latitude,longitude format.

+ (id)distance:(NSArray *)params

Parameters

params

an array containing the input parameters for the function

Return Value

An NSNumber instance containing a double value representing the distance in miles between the input parameters

Discussion

Template example:

^distance($pointA|$pointB)

The expression above returns the distance in miles between $pointA and $pointB.

+ formatDistance:

Calculates the distance between two map coordinates, in miles, and formats a string to represent that distance.

+ (id)formatDistance:(NSArray *)params

Parameters

params

an array containing the input parameters for the function

Return Value

An NSString formatted according to the parameters

Discussion

This function accepts an array of template expressions, and expects three objects in the array:

  • a format string
  • point a and point b, both of which are expected to evaluate to MBGeolocation instances or NSStrings. If either point is a string, an attempt is made to convert it into an MBGeolocation instance by interpreting it in latitude,longitude format.

Template example:

^formatDistance(%0.1f miles|$pointA|$pointB)

The expression above would determine the miles between $pointA and $pointB, and would then format that value into a string using [NSString stringWithFormat:@“%0.1f miles”, distanceInMiles].

+ parseLocation:

Attempts to create an MBGeolocation instance by parsing a string for decimal latitude and longitude coordinates.

+ (id)parseLocation:(NSString *)locStr

Parameters

a

string containing comma-separated or pipe-separated latitude and longitude coordinates

Return Value

An MBGeolocation containing the latitude and longitude specified in the input parameter

Discussion

This function accepts a string as input, where the string contains the latitude coordinate, followed by a delimiter, followed by the longitude coordinate.

This method accepts either a comma (“,”) or a pipe character (“|”) as the delimiter, meaning that the following expressions are equivalent:

^parseLocation(40.741284|-73.987383)
^parseLocation(40.741284,-73.987383)

Either expression above would return an MBGeolocation instance pointing to the address 1 Madison Avenue in New York City.
